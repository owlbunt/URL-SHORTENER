API Documentation

Base URL

https://s.harishhonda.com/api/

Generate a Short URL

To generate a short URL with a random alias, send a GET request to:

https://s.harishhonda.com/api/<longurl>

Example:

https://s.harishhonda.com/api/https://example.com

Response:

{"short_url": "https://s.harishhonda.com/xyz123"}

Generate a Short URL with Custom Alias

To create a short URL with a custom alias, use the alias query parameter:

https://s.harishhonda.com/api/<longurl>?alias=<custom_name>

Example:

https://s.harishhonda.com/api/https://example.com?alias=myalias

Response:

{"short_url": "https://s.harishhonda.com/myalias"}

Response Format

All responses are returned in JSON format.

Errors

In case of an error, the API will return a JSON response with an error message.

{"error": "Invalid URL"}
